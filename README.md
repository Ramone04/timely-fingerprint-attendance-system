# Timely - Fingerprint Attendance System

## 📌 Overview
Sistema de controlo de assiduidade com ESP32 e sensor biométrico.

## 🧰 Hardware
- ESP32
- R307 Fingerprint
- LCD I2C

## ⚙️ Funcionalidades
- Enroll de utilizadores
- Registo de entradas/saídas
- Comunicação MQTT

## 🚀 Futuro
- Abertura de porta com biometria
